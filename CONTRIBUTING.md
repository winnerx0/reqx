# Contributing to reqx

First off, thanks for taking the time to contribute!

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
   ```bash
   git checkout -b my-new-feature
   ```
3. Make your changes types.
4. Commit your changes.
   ```bash
   git commit -m 'Add some feature'
   ```
5. Push to the branch.
   ```bash
   git push origin my-new-feature
   ```
6. Open a Pull Request.

## Reporting Bugs

If you find a bug, please create a new issue on the repository with details on how to reproduce it.

## Suggesting Enhancements

We welcome suggestions for new features or improvements. Please open an issue to discuss your ideas.
